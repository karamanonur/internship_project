This app provides information about locations and names of tram stops in Eski≈üehir, map center changes when clicking on tram stop names. Also one can see concerts, exhibitions, events and etc. which will occur in town, this information is get via an api which is also written by us. These event informations are added to a postgresql database via an admin panel. Finally, the closest tram stop to user's current location can be seen.
